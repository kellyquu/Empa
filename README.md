## Getting Started

### Starting the server

_(127.0.0.1:5000 by default)_

1. `cd server`
1. `python3 -m venv venv`
1. `source venv/bin/activate` (MacOS)
1. `venv\Scripts\activate` (Windows Powershell)
1. `pip install -r requirements.txt`
1. `python3 app.py`

### Starting the app

_(localhost:3000 by default)_

1. `cd client`
1. `npm install`
1. `npm start`
